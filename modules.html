beam

Beam definition and cross-section helper functions.

@author: Vincent Mahé

class beamsolve.beam.beam.Beam(L, E, rho, A, I, bc_left='clamped', bc_right='free')[source]

Bases: object

An Euler-Bernoulli beam.

The beam is characterised by its geometry, material properties, and boundary conditions at both ends.

Parameters:

  • L (float) – Beam length [m].

  • E (float) – Young’s modulus [Pa].

  • rho (float) – Material density [kg/m³].

  • A (float) – Cross-sectional area [m²].

  • I (float) – Second moment of area [m⁴].

  • bc_left ({"clamped", "free", "pinned"}) – Boundary condition at the left end (\(x = 0\)).

  • bc_right ({"clamped", "free", "pinned"}) – Boundary condition at the right end (\(x = L\)).

Examples

Clamped-free beam with a rectangular cross-section:

>>> from beamsolve.beam import Beam, rectangular_section
>>> A, I = rectangular_section(b=0.02, h=0.005)
>>> beam = Beam(L=1.0, E=210e9, rho=7800, A=A, I=I,
...             bc_left="clamped", bc_right="free")

Simply supported beam with a circular cross-section:

>>> from beamsolve.beam import Beam, circular_section
>>> A, I = circular_section(d=0.01)
>>> beam = Beam(L=1.0, E=70e9, rho=2700, A=A, I=I,
...             bc_left="pinned", bc_right="pinned")

class beamsolve.FEM.fem_model.FEMModel(beam, ne=20)[source]

Bases: object

Finite element model of an Euler-Bernoulli beam.

The beam is discretised into ne elements of equal length. Each node has two degrees of freedom: transverse displacement \(v\) and rotation \(\theta\). The global stiffness matrix \(\mathrm{K}\) and mass matrix \(\mathrm{M}\) are assembled using the standard Hermite shape functions.

Parameters:

  • beam (Beam) – The beam to model.

  • ne (int) – Number of finite elements.

Examples

>>> from beamsolve.beam import Beam, rectangular_section
>>> from beamsolve.FEM import FEMModel
>>> A, I = rectangular_section(b=0.02, h=0.005)
>>> beam = Beam(L=1.0, E=210e9, rho=7800, A=A, I=I,
...             bc_left="clamped", bc_right="free")
>>> model = FEMModel(beam=beam, ne=20)

class beamsolve.Analytical.analytical_model.AnalyticalModel(beam, n_modes=5)[source]

Bases: object

Analytical modal analysis of an Euler-Bernoulli beam.

Parameters:

  • beam (Beam) – The beam to analyse.

  • n_modes (int) – Number of modes to compute.

Notes

The natural frequencies and mode shapes are computed from the analytical solution of the Euler-Bernoulli beam equation

\[EI \frac{\partial^4 V}{\partial x^4} - \rho A \omega^2 V = 0.\]

The general solution for the mode shape \(V(x)\) is

\[V(x) = C_1 \cos(\beta x) + C_2 \sin(\beta x) + C_3 \cosh(\beta x) + C_4 \sinh(\beta x),\]

where the wavenumber \(\beta\) is related to the angular frequency by

\[\beta^4 = \frac{\rho A \omega^2}{EI}.\]

The boundary conditions at both ends are gathered in a \(4 \times 4\) matrix \(\mathrm{Z}(\beta)\) such that \(\mathrm{Z}(\beta)\,\mathbf{C} = \mathbf{0}\). Natural frequencies correspond to values of \(\beta\) for which \(\det \mathrm{Z}(\beta) = 0\). The associated mode shape coefficients \(\mathbf{C}\) are obtained as the null vector of \(\mathrm{Z}(\beta)\) via SVD.

The rows of \(\mathrm{Z}(\beta)\) encode the following conditions:

  • Clamped end: \(V = 0\) and \(V' = 0\)

  • Pinned end: \(V = 0\) and \(V'' = 0\)

  • Free end: \(V'' = 0\) and \(V''' = 0\)

Examples

>>> from beamsolve.beam import Beam, rectangular_section
>>> from beamsolve.Analytical import AnalyticalModel
>>> A, I = rectangular_section(b=0.02, h=0.005)
>>> beam  = Beam(L=1.0, E=210e9, rho=7800, A=A, I=I,
...              bc_left="clamped", bc_right="free")
>>> analytical_model = AnalyticalModel(beam=beam, n_modes=3)
>>> sol   = analytical_model.solve_modes()
>>> sol.frequencies

bc_rows(bc, x, beta)[source]

Return the two rows of \(\mathrm{Z}(\beta)\) corresponding to boundary condition bc applied at position x.

Parameters:

  • bc (str) – Boundary condition type ("clamped", "pinned", "free").

  • x (float) – Position [m] where the BC is applied.

  • beta (float) – Wavenumber [1/m].

Notes

The mode shape \(V(x) = C_1\cos(\beta x) + C_2\sin(\beta x) + C_3\cosh(\beta x) + C_4\sinh(\beta x)\) and its derivatives evaluated at x give the following rows:

\[\begin{split}V &: [\cos(\beta x),\ \sin(\beta x),\ \cosh(\beta x),\ \sinh(\beta x)] \\ V' &: \beta\,[-\sin(\beta x),\ \cos(\beta x),\ \sinh(\beta x),\ \cosh(\beta x)] \\ V'' &: \beta^2\,[-\cos(\beta x),\ -\sin(\beta x),\ \cosh(\beta x),\ \sinh(\beta x)] \\ V''' &: \beta^3\,[\sin(\beta x),\ -\cos(\beta x),\ \sinh(\beta x),\ \cosh(\beta x)]\end{split}\]
Returns:

rows – Two rows of the BC matrix.

Return type:

numpy.ndarray, shape (2, 4)

mode_coefficients(beta)[source]

Compute the mode shape coefficients \(\mathbf{C} = [C_1, C_2, C_3, C_4]\) for a given wavenumber via SVD of \(\mathrm{Z}(\beta)\).

At a root \(\beta_i\), \(\mathrm{Z}(\beta_i)\) has rank 3 and its smallest singular value is (numerically) zero. The associated right singular vector — the last row of \(\mathrm{V}^T\) in the decomposition \(\mathrm{Z} = \mathrm{U} \mathrm{\Sigma} \mathrm{V}^T\) — spans the null space and gives \(\mathbf{C}\).

Parameters:

beta (float) – Wavenumber [1/m].

Returns:

C – Mode shape coefficients \([C_1, C_2, C_3, C_4]\).

Return type:

numpy.ndarray, shape (4,)
