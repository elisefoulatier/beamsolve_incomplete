Simply-supported beam

Modal analysis of a simply-supported aluminium beam: FEM vs analytical solution.

System description

The beam is supported at \(x = 0\) (displacement zero) and supported at \(x = L\) (displacement y zero). It has length \(L\), Young’s modulus \(E\), density \(\rho\), and a circular cross-section of diameter \(d\).

The analytical characteristic equation for a simply-supported beam is

\[1 + \cosh(\beta L)\cos(\beta L) = 0.\]

Code description

The script below

  • Constructs the beam using beamsolve.beam.beam.circular_section() and beamsolve.beam.beam.Beam.

  • Solves the eigenvalue problem using beamsolve.FEM.fem_model.FEMModel and beamsolve.FEM.fem_model.FEMModel.solve_modes().

  • Computes the exact solution using beamsolve.Analytical.analytical_model.AnalyticalModel.

  • Compares FEM and analytical natural frequencies.

  • Plots both sets of mode shapes.
